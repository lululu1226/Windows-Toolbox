## 功能一览
- 硬件信息检测：CPU / 主板 / 内存 / 硬盘 / 显卡 / 电池 / 网卡
- 性能测试：屏幕坏点检测、硬盘/U盘测速、烤机测试、整机跑分
- 系统优化：垃圾清理、启动项管理、网络修复、端口扫描
- 实用工具：电源模式切换、局域网扫描、文件哈希校验
- 扩展功能：卸载预装应用、清除系统历史记录

## 运行方式
1. 安装依赖：
   pip install ttkbootstrap psutil cpuinfo GPUtil wmi
2. 以管理员身份运行：
   python Windows工具箱.py
